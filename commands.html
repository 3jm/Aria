Commands

[ ] - is a OPTIONAL argument
( ) - is a REQUIRED argument

Command Description Required Permissions
help Directs you to this page None
info Displays information about the bot None
ping Displays the bots ping None
myserver Displays information about your server None
whois [member] Displays information about a server member None
bitcoin Displays the current bitcoin price in EUR & USD None
joined [member] Displays when a user joined the server None
neko captcha (Image URL) (Username) Displays a captcha based off a image & username None
neko threats (Image URL) Displays the top 3 biggest threats to society None
neko fact (Text) Displays a fact None
